# Private Events

> A site to create events.

<p align="center">
    <img src="page.png">
</p>


The porpuse of this project was to build a site similar to a private [Eventbrite](https://www.eventbrite.com/) which allows users to create events and then manage user signups.

## Built With

- Ruby on Rails 5.2.4
